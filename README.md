# 🛍️ ROOMナビ — 楽天ROOMアフィリ支援アプリ

楽天ROOMで稼ぐための「投稿ボタンを押す直前まで」を自動化する単一HTMLアプリ。
リサーチ（何を載せるか）→ 投稿文生成 → ストック管理 → 実績記録 を1つにまとめています。

> **なぜ全自動投稿ではないのか**：楽天ROOMには公開の投稿APIがなく、
> 自動投稿ボットは利用規約違反（発覚するとアカウント停止＝収益ゼロ）です。
> そのため本アプリは規約の範囲内で、投稿以外の全工程を自動化します。

## 使い方（最初の1回だけ）

1. `index.html` をダブルクリックで開く（ビルド・インストール不要）
2. 「⚙️設定」タブで楽天APIキーを保存
   - **アプリID（必須）**: [楽天ウェブサービス](https://webservice.rakuten.co.jp/) にログイン→「アプリID発行」（無料・即時）
   - **アフィリエイトID**: [楽天アフィリエイト](https://affiliate.rakuten.co.jp/) の管理画面に表示（例 `1a2b3c4d.5e6f7a8b`）
   - ※アフィリエイトIDを入れないと報酬が発生するリンクになりません（アプリが警告します）

## 毎日の流れ（1回10分）

1. **🔍リサーチ**: キーワード検索 or ジャンル別ランキングで商品を取得。
   「スコア順」= 1件あたり報酬(円) × 売れ筋度(レビュー数・評価) × 売れやすい価格帯(1,000〜9,999円)ボーナス。
   上位から「✍️投稿文を作る」を押すだけ。
2. **✍️投稿文**: テンプレ4種（悩み解決/愛用レビュー風/比較/季節）＋ハッシュタグで自動生成。
   もっと自然な文が欲しい時は「🤖AI用プロンプトをコピー」→ Claude に貼ると3案出ます。
3. **投稿**: 「📋コピー」→「🔗ROOMを開く」→ ROOMで商品を検索して「コレ!」→ コメント欄にペースト。
   目安は1日3〜5投稿。
4. **📦ストック**: 書き溜めて下書き→投稿済みを管理。今日の投稿数も表示。
5. **📈実績**: 楽天アフィリエイトのレポートからクリック/成果/報酬を転記→月次集計とグラフ。

## データについて

- 保存は端末内の localStorage のみ（キー `roomNavi_v1`）。外部送信なし（楽天APIへの検索を除く）
- バックアップ: 設定/ストックタブから JSON 書き出し・取り込み（**冪等**＝同じファイルを何度取り込んでも重複しない）
- オフラインでも投稿文生成・ストック・実績は動作（商品検索のみネット必要）

## 自動テスト

```
cd C:\Users\user\rakuten-room-navi
npm test
```

Node標準の `node --test` で 37テスト。`index.html` 内の `/*CORE_START*/〜/*CORE_END*/` の
純ロジック（`RoomCore`）を抽出して検証します（URL構築 / API応答の正規化 / 報酬スコア計算 /
投稿文生成 / 冪等マージ / 実績集計 など）。**index.html のコアを直したら必ず `npm test` を通すこと。**

## 制限・注意

- 楽天APIは約1リクエスト/秒の制限あり。連打すると `too_many_requests` になる（少し待てばOK）
- 投稿文の誇大表現（「絶対痩せる」等）は薬機法・景表法NG。テンプレとAIプロンプトはNG表現を避ける設計
- Netlify等に公開する場合もAPIキーは各端末のlocalStorageに保存される（HTMLには埋め込まれない）
